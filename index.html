Soul Tax

Game running slow? Make sure you have the latest Flash Player!
Info
  • Made in one month for the Something Awful Game Dev VI event.
  • Developed using the Flixel framework.
  • Feel free to check out or contact any of the developers: Jarod Long, Lauren Careccia, Brad Snyder.

Controls

  • ESDF or WASD: Move/Jump
  • Space: Possess/Unpossess
  • J: Punch
  • K: Kick
  • L: Special ability
